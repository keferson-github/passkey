# Projeto PassKey
## Trata-se de um sistema de gerenciamento e criação de senhas para uso pessoal

- React
- TypeScript
- Tailwind.CSS
- Shadcn/ui
- Autenticação de Usuários
- Banco de Dados PostgreSQL
